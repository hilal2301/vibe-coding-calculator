## Challenges and Solutions
One challenge I encountered was ensuring the mathematical logic correctly handled sequential operations. Initially, the AI used a simple evaluation method, but I had to guide it to handle edge cases like "division by zero" more gracefully. v0.dev helped by quickly refactoring the calculation logic when I pointed out these bugs.

## Reflection
What surprised me most about "vibe coding" was how little manual syntax writing was required. The focus shifted from "how to write the loop" to "how the application should behave." This technology changes software development by making it more about architecture and intent rather than just typing code. I would definitely use this tool again for front-end prototyping.
