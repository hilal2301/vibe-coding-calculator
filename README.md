# AI-Powered Scientific Calculator

A functional scientific calculator built using the "Vibe Coding" paradigm.

## Features
* **Scientific Functions:** Support for sin, cos, tan, log, and square root.
* **History Sidebar:** Keeps track of previous calculations for quick reference.
* **Modern UI:** Dark mode support with a sleek glassmorphism design.
* **Responsive:** Fully functional on both desktop and mobile devices.

## Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **AI Tool:** v0.dev

## Setup and Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:3000` in your browser.
